## ⚠️ 重要声明

1. **本工具仅供学习研究使用**，请勿用于违反飞书用户协议的场景。
2. RPA 方式模拟真人操作，响应速度通常为 **1-3 秒**，无法做到 API 级别的毫秒级响应。
3. 飞书网页版 DOM 结构可能随时更新，如遇到选择器失效，请提交 Issue。
4. 建议保持默认的反检测延迟设置，避免触发风控。

## 📥 安装与使用

### 方式一：下载可执行文件（推荐，无需 Python 环境）

1. 前往 [Releases 页面](https://github.com/m0ga0/feishu-auto-liker-gui/releases) 下载最新版本
2. 解压后双击运行 `飞书自动点赞助手.exe`（Windows）或对应平台程序
3. 首次运行会自动安装依赖，点击 **"一键安装"** 按钮即可
4. 安装完成后，切换到 **"控制台"** 标签页，点击 **"启动监控"**

### 方式二：从源码运行（需要 Python 环境）

```bash
# 1. 克隆项目
git clone https://github.com/m0ga0/feishu-auto-liker-gui.git
cd feishu-auto-liker-gui

# 2. 安装依赖
pip install -r requirements.txt
playwright install chromium

# 3. 运行
python main.py
```

## 🖥️ 界面说明

软件包含三个标签页：

### 1️⃣ 安装

首次使用时进入此页面，点击 **"🚀 一键安装"** 按钮：

- ✅ 自动检查 Python 环境
- ✅ 自动安装 Playwright 库
- ✅ 自动下载 Chromium 浏览器
- 📋 实时显示安装进度和日志

安装完成后，即可切换到其他标签页使用。

### 2️⃣ 控制台

主操作界面，包含：

- **状态指示器** — 显示运行状态（🟢 运行中 / ⏹ 已停止）
- **实时统计** — 匹配次数、点赞成功、失败次数、运行时长
- **控制按钮** — 启动监控、停止、重置统计
- **运行日志** — 实时显示监控日志

### 3️⃣ 设置

配置所有运行参数：

| 设置项 | 说明 |
|--------|------|
| 🎯 监控关键词 | 每行一个，支持 `re:` 前缀的正则表达式 |
| 😀 点赞表情 | 选择要自动点击的表情 |
| 💬 监控群组 | 每行一个群名，留空表示监控当前活跃聊天 |
| ⏱ 检查间隔 | 轮询频率（秒），默认 2 秒 |
| 🛡️ 反检测延迟 | 模拟人类操作的随机延迟范围 |
| 🔔 通知设置 | 桌面弹窗通知、发送消息到文件助手 |

修改后点击 **"💾 保存设置"** 即可生效。

## 📦 自行打包为可执行文件

如果你需要从源码打包为独立的可执行文件：

```bash
# 安装打包工具
pip install pyinstaller

# 运行打包脚本
python build.py

# 打包完成后，可执行文件位于 dist/ 目录
```

**注意**：打包后的程序体积较大（约 200-300MB），因为包含了 Chromium 浏览器以外的所有依赖。Chromium 浏览器仍需在首次运行时自动下载。

## 🔧 常见问题

### Q: 首次运行卡住在"等待登录"？
A: 请在自动打开的浏览器中完成飞书登录（扫码或输入账号密码）。登录成功后软件会自动继续。

### Q: 提示"选择器失效"或无法找到消息？
A: 飞书网页版更新后 DOM 结构可能变化。请尝试：
1. 更新到最新版本
2. 如仍未修复，请提交 Issue 并附上飞书网页版的截图

### Q: 能否在后台运行？
A: 不建议开启无头模式（headless），因为飞书可能检测到并拒绝登录。建议保持浏览器窗口打开。

### Q: 登录状态能保持多久？
A: 通常可以保持数天到数周。如果提示重新登录，只需在浏览器中重新登录即可。
